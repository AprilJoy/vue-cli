# 文档说明

## 使用方法
开发模式
1.启动vue前端

```
进入根目录
yarn serve
```

2启动express 后端

```
cd api
nodemon server
```

3.启动mongoDB

```
mongod --dbpath /usr/local/var/mongodb --port 8900
```

## clone代码后需要执行
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
